# Алгоритмы и структуры данных
### Вторая  неделя: <b>Списки</b>


1. <b>Проверить, является ли список циклическим</b>
    Дан односвязный список. Необходимо проверить, является ли этот список циклическим.

2. <b>Развернуть односвязный список</b>
    Необходимо написать функцию, которая принимает на вход односвязный список и разворачивает его.

3. <b>Найти середину списка</b>
    Дан связный список. Необходимо найти середину списка. Сделать это необходимо за O(n) без дополнительных аллокаций.

4. <b>Удалить элемент из односвязного списка</b>
    Необходимо написать функцию, которая принимает на вход односвязный список и некоторое целое число val. Необходимо удалить узел из списка, значение которого равно val.

5. <b>(HM) Слияние двух отсортированных списков</b>
    Написать функцию, которая принимает на вход два отсортированных односвязных списка и объединяет их в один отсортированный список. При этом затраты по памяти должны быть O(1)
